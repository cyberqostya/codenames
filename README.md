## О приложении

Это мобильная версия игры в духе Codenames, адаптированная под игру с одного телефона. Приложение можно открыть как обычный сайт или как Telegram Mini App.

Игроки делятся на команды, получают поле карточек и по очереди пытаются открыть свои карточки, не попав на черную. Один из игроков может включить режим капитана, чтобы увидеть карту принадлежности карточек командам.

## Возможности для игроков

- Несколько наборов карточек: слова, стандартные картинки, AI-картинки и смешанный режим.
- Настройка размера игрового поля через количество колонок и строк.
- Случайная генерация карточек и капитанской карты для новой партии.
- Отдельная кнопка для смены только капитанской карты без перемешивания карточек.
- Отдельная кнопка для полного перемешивания карточек.
- Режим капитана с коротким обратным отсчетом, звуком и анимацией, чтобы игроки не увидели карту случайно.
- Долгое удержание карточки для открытия, чтобы защититься от случайных тапов.
- Быстрый тап по карточке-картинке открывает увеличенный просмотр.
- Возможность редактировать текстовые карточки прямо в настройках.
- Защита от неудобной ориентации экрана: приложение просит вернуть телефон в портретный режим.
- Вибрация на действиях в Telegram Mini App.
- Подготовка оффлайн-режима: приложение кеширует сборку и ассеты, чтобы после загрузки можно было играть без сети там, где браузер или WebView поддерживает Service Worker.

## Техническая часть

- Vue 3 Composition API, Vite и Pinia.
- Деплой на GitHub Pages через GitHub Actions без хранения `dist` в репозитории.
- Автоматическая генерация `sw.js` во время production build: service worker получает актуальный список файлов из `dist`, поэтому новые ассеты не нужно вручную добавлять в кеш.
- Отдельный слой контента: слова и изображения импортируются как данные, а не хардкодятся в компонентах.
- Автоматический импорт изображений через `import.meta.glob`, без привязки игровой логики к именам `pic1`, `pic2` и так далее.
- Адаптивное игровое поле на CSS Grid с динамическим количеством колонок.
- Отдельные Vue-компоненты для доски, карточки, настроек, редактора слов, просмотра картинки, защиты ориентации и капитанского отсчета.
- Интеграция с Telegram WebApp API: инициализация Mini App, расширение окна, отключение вертикальных свайпов, haptic feedback.
- Аккуратная обработка тач-сценариев: быстрый тап, долгий тап, запрет контекстного меню и защита от случайного открытия карточки.
- Использование Web Audio API для короткого звукового сигнала при отсчете.
- Оффлайн-статус в интерфейсе настроек с отображением процесса кеширования.
